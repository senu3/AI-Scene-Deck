インタラクション仕様

シーンチップバー

  • クリック - そのシーンを選択（ストーリーラインがスクロールしてフォーカス）
  • ダブルクリック - シーン名を編集
  • 右クリック - コンテキストメニュー（名前変更/複製/削除）
  • ドラッグ - シーンの並び替え
  • ホバー - ツールチップでシーン詳細を表示
  • Ctrl+クリック - 複数シーン選択

フッター

  • PREVIEW - 全体プレビューモーダルを開く
  • EXPORT - エクスポートモーダルを開く
  • 統計はリアルタイム更新（シーン/カット追加削除に連動）

キーボードショートカット（提案）

  • ← / → - 前/次のシーンを選択
  • Home / End - 最初/最後のシーンを選択
  • Ctrl+← / Ctrl+→ - 選択シーンを左/右に移動
  • Space - プレビュー開始
  • Ctrl+E - エクスポート
